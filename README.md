This repo contains an app that demonstrates a bug in the macOS keychain APIs. The `SecItemUpdate()` function fails randomly with `errSecItemNotFound` (-25300). After generating a keypair, `SecItemUpdate()` will succeed a few times. Continuing to call `SecItemUpdate()` will result in many failures intermixed with sporadic successes.

Here's the output from the keychaintest app:

```
sleeping
sleeping
Successfully set attr
0
Successfully set attr
1
Error from keychain d: -25300
deleted: true
```
